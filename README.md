## 🚀 How to Use

1️⃣ **Extract** the archive to any folder  
2️⃣ **Run** `ProductionSuite.exe` as Administrator 🛡️  
3️⃣ **Select** your Ableton Live install directory  
4️⃣ **Pick** a preset (Electronic / Hip-Hop / Ambient / Mastering) or tweak each module  
5️⃣ **Hit Apply** — done! 🎉  
6️⃣ **Launch** Ableton Live and produce without limits

> 💡 **Pro tip:** Run **CPU Optimizer** first, then **VST Manager** to disable problematic plugins before starting a session.

## ❗ Troubleshooting

| ❌ Problem | ✅ Fix |
|---|---|
| CPU still peaking | Freeze tracks, increase buffer size via CPU Optimizer |
| VST not detected | Rescan plugin folders in VST Manager |
| Export stems silence | Check track routing, ensure solo/mute states are cleared |
| MIDI latency | Adjust compensation in MIDI Tuner, check audio interface drivers |
| Sample preview silent | Verify audio output device in Sample Librarian settings |
| Template corrupted | Reload from Template Manager backup folder |
